# ApexPlanet - Web Development Internship Tasks 🚀

Welcome to my repository for the 45-day Web Development internship program at ApexPlanet Software Pvt Ltd. This repository contains all the frontend assignments and projects built using HTML, CSS, and JavaScript.

## 📂 Repository Structure & Task Overview

The internship program is divided into 5 modules. Here is a quick overview of what each task covers:

- **[Task-1](./Task-1): Basics of Web Development**
  Built a foundational webpage using HTML, styled it with CSS, and added basic JavaScript interactivity.

- **[Task-2](./Task-2): Intermediate HTML, CSS, and JavaScript**
  Created a responsive layout with a contact form utilizing JS validation and dynamic DOM manipulation.

- **[Task-3](./Task-3): Advanced Styling and JavaScript**
  Implemented advanced media queries, built interactive components, and fetched real-time data from an external API.

- **[Task-4](./Task-4): Full Project Implementation**
  Developed a personal portfolio website emphasizing modern layout designs, a local-storage based app, and a filtered product page.

- **[Task-5](./Task-5): Final Project and Optimization**
  Built a comprehensive capstone web application, optimizing for loading performance and ensuring cross-browser compatibility.

## 🛠️ Technologies Used

- **Markup & Styling:** HTML5, CSS3
- **Scripting:** Vanilla JavaScript (ES6+)
- **Version Control:** Git, GitHub

## 🚀 How to Run Locally

Since these are pure static frontend projects, no heavy dependencies or build tools are required:

1. **Clone this repository:**
   ```bash
   git clone https://github.com/dbaidya811/ApexPlanet.git
   ```

2. **Open the project:**
* Navigate to the specific task folder (e.g., `Task-1`).
* Simply double-click the `index.html` file to open it in your default web browser.
